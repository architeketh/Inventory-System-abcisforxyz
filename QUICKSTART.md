## Backing Up Your Data

### Quick Backup (30 seconds)
1. Go to "Data Management" tab
2. Click "Export to JSON"
3. Upload the file to your GitHub repository

### Quick Restore (30 seconds)
1. Download JSON file from GitHub
2. Go to "Data Management" tab
3. Select file and click "Import from JSON"

## Testing with Sample Data
